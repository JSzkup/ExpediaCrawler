This program opens up a Chrome webview window and automates searching for a flight from one specified place to another at specified dates.
It then maps the results in an EXCEL doc, saves it then emails a specified email the flight info on the flight witht the cheapest price.

The program is set to run once an hour for 8 hours. Thats 8 emails, with 8 flights.


You must download Webdriver from googles website for your correct version of Chrome
Place the .exe the main Chrome Directory

All credentials for your email & Expedia are to be inputted at the bottom of the program

If program does not run normally, run in DEBUG MODE

Dependencies (pip install - ):
Selenium
Pandas
openpyxl
